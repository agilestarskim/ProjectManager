# Project Manager

## ☘️ 소개
칸반보드를 이용한 프로젝트 관리 어플리케이션입니다.  
할 일 목록을 만들고 삭제하고 수정할 수 있습니다.  
모든 활동 내역은 로컬 데이터베이스와 리모트 데이터베이스에 저장됩니다.  
간단한 로그인을 통해 계정별로 데이터를 관리할 수 있습니다.  

</br>

## 🛠️ 기술

`SwiftUI` `Realm` `Firebase` `Async/Await` `클린 아키텍처 + MVVM`


</br>
